Básica
IF — La función más usada en Excel
Evalúa una condición y devuelve un valor si es verdadera y otro si es falsa. La base de toda lógica en Excel.
SINTAXIS
=IF(condición, valor_si_verdadero, valor_si_falso)
condición
Expresión que se evalúa como TRUE o FALSE. Ej: A2>1000, B2="Lima"
si_verdadero
Qué devolver si la condición es TRUE
si_falso
Qué devolver si la condición es FALSE
Caso 1 — Categorizar ventas
=IF(D2>=5000,"Alta","Baja")
Si Total ≥ S/5000 → "Alta", si no → "Baja"
→ "Alta" si D2=7000
Caso 2 — Detectar meta cumplida
=IF(C2>=B2,"✓ Meta","✗ Meta")
Compara ventas reales vs meta
→ "✓ Meta" si supera la meta
IF anidado
Para múltiples condiciones: =IF(D2>=10000,"Premium",IF(D2>=5000,"Media","Baja")). Puedes anidar hasta 64 niveles, pero más de 3 se vuelve ilegible — usa IFS() en su lugar.
Intermedia
IFS — IF múltiple sin anidar
Evalúa múltiples condiciones en orden. Más limpio que anidar IFs cuando tienes 3+ categorías.
SINTAXIS
=IFS(cond1, val1, cond2, val2, ...)
Caso real — Clasificar performance de vendedor
=IFS(E2>=95,"Excelente", E2>=80,"Bueno", E2>=60,"Regular", E2<60,"Bajo")
→ Evalúa en orden: primera condición TRUE gana
Ojo con el orden
IFS evalúa de izquierda a derecha y se detiene en la primera condición TRUE. Siempre pon las condiciones de mayor a menor (≥95 antes que ≥80).
Intermedia
AND / OR — Condiciones compuestas
Se usan dentro de IF para evaluar múltiples condiciones a la vez.
AND — todas deben cumplirse
=IF(AND(B2="Lima",C2>5000),"Prioridad","Normal")
Solo "Prioridad" si es Lima Y ventas >5000
OR — basta con una
=IF(OR(B2="Lima",B2="Norte"),"Zona A","Zona B")
"Zona A" si es Lima O Norte
Intermedia
VLOOKUP — Búsqueda vertical clásica
Busca un valor en la primera columna de una tabla y devuelve el valor de otra columna en la misma fila. La función de búsqueda más pedida en entrevistas.
SINTAXIS
=VLOOKUP(valor_buscado, tabla, núm_columna, 0)
valor_buscado
Lo que buscas. Ej: el ID del vendedor
tabla
El rango donde buscas. La primera columna DEBE contener el valor buscado
núm_columna
Qué columna de la tabla quieres que devuelva (1=primera, 2=segunda...)
0 (FALSE)
Siempre 0 para búsqueda exacta. Con 1 busca aproximada — casi nunca lo usas
Caso real — Traer el nombre del vendedor por su ID
=VLOOKUP(A2, Vendedores!$A:$C, 2, 0)
Busca el ID en columna A de la hoja Vendedores, devuelve la columna 2 (Nombre)
→ Devuelve "Ana Torres" si A2="V-001"
La limitación más importante de VLOOKUP
Solo busca hacia la derecha — el valor buscado debe estar en la primera columna de tu rango. Si necesitas buscar hacia la izquierda, VLOOKUP no puede. Para eso existe XLOOKUP o INDEX+MATCH.
Avanzada
XLOOKUP — El reemplazo moderno de VLOOKUP
Más flexible, más poderoso y más legible que VLOOKUP. Busca en cualquier dirección, maneja errores integrado, y no necesita contar columnas.
SINTAXIS
=XLOOKUP(valor_buscado, rango_búsqueda, rango_retorno, [si_no_encontrado])
valor_buscado
Lo que buscas
rango_búsqueda
Columna donde buscar (no toda la tabla, solo la columna)
rango_retorno
Columna que quieres que devuelva
si_no_encontrado
Qué mostrar si no encuentra nada. Ej: "No encontrado"
Mismo caso — más limpio
=XLOOKUP(A2, Vendedores!A:A, Vendedores!B:B, "N/A")
→ No necesitas contar columnas
Buscar hacia la izquierda
=XLOOKUP(B2, Vendedores!B:B, Vendedores!A:A, "N/A")
→ Busca Nombre, devuelve ID (imposible en VLOOKUP)
¿VLOOKUP o XLOOKUP?
Si trabajas en Excel 365 u Office 2021+, usa siempre XLOOKUP. Si trabajas con archivos compartidos con personas en versiones antiguas de Excel, usa VLOOKUP para compatibilidad.
Avanzada
INDEX + MATCH — La combinación profesional
La alternativa clásica a VLOOKUP que funciona en todas las versiones. MATCH encuentra la posición, INDEX devuelve el valor en esa posición. Juntas son imbatibles.
SINTAXIS COMBINADA
=INDEX(columna_retorno, MATCH(valor_buscado, columna_búsqueda, 0))
Equivalente al VLOOKUP anterior
=INDEX(Vendedores!B:B, MATCH(A2, Vendedores!A:A, 0))
MATCH encuentra en qué fila está A2 dentro de la columna A de Vendedores. INDEX devuelve el valor de la columna B en esa fila.
→ Funciona en todas las versiones de Excel, busca en ambas direcciones

Básica
SUMIF / SUMIFS — Suma con condiciones
Suma solo las filas que cumplen una o más condiciones. Una de las más usadas en reportes de negocio.
SINTAXIS
=SUMIF(rango_criterio, criterio, rango_suma)
=SUMIFS(rango_suma, rango1, criterio1, rango2, criterio2, ...)
SUMIF — una condición
=SUMIF(B:B,"Lima",E:E)
Suma la columna E solo donde B="Lima"
→ Total ventas de Lima
SUMIFS — dos condiciones
=SUMIFS(E:E,B:B,"Lima",C:C,"Electrónica")
Suma E donde B=Lima Y C=Electrónica
→ Ventas Lima + Electrónica
Básica
COUNTIF / COUNTIFS — Contar con condiciones
Cuenta cuántas filas cumplen una o más condiciones. Esencial para frecuencias, auditorías y KPIs.
¿Cuántas ventas hubo en Lima?
=COUNTIF(B:B,"Lima")
→ Número entero: cantidad de filas
¿Cuántas ventas >S/1000 en Norte?
=COUNTIFS(B:B,"Norte",E:E,">1000")
→ Cuenta con dos filtros
AVERAGEIF también existe
=AVERAGEIF(B:B,"Lima",E:E) calcula el promedio de ventas solo para Lima. Misma lógica que SUMIF pero promedia en vez de sumar.

Funciones de Texto más usadas
LEFT / RIGHT / MID — extraer texto
=LEFT(A2,3) → "V-0" de "V-001" =RIGHT(A2,3) → "001" de "V-001" =MID(A2,3,3) → "001" desde pos 3
CONCATENATE / & — unir texto
=A2&" "&B2 =CONCAT(A2," ",B2)
→ "Ana Torres Lima"
FIND / SEARCH — posición de texto
=FIND("-",A2) → posición del guion
FIND es sensible a mayúsculas, SEARCH no
TEXT — formatear números como texto
=TEXT(D2,"S/ #,##0.00")
→ "S/ 5,000.00" para reportes
Intermedia
Funciones de Fecha más usadas
Extraer partes de fecha
=YEAR(A2) → 2024 =MONTH(A2) → 1 (enero) =DAY(A2) → 5
Calcular diferencias
=DATEDIF(A2,B2,"D") → días entre fechas =DATEDIF(A2,B2,"M") → meses =TODAY()-A2 → días desde hoy
Avanzada
Fórmulas anidadas — el nivel profesional
Combinar funciones dentro de otras es lo que diferencia un usuario básico de Excel de un analista. Estos son los patrones más usados en el trabajo real.
Patrón 1 — IFERROR + VLOOKUP (manejo de errores)
=IFERROR(VLOOKUP(A2,Tabla!$A:$C,2,0),"No encontrado")
Si VLOOKUP no encuentra el valor (error #N/A), muestra "No encontrado" en vez de un error feo.
Patrón 2 — IF + SUMIFS (KPI dinámico)
=IF(SUMIFS(E:E,B:B,"Lima")>=100000,"✓ Meta","✗ "&TEXT(SUMIFS(E:E,B:B,"Lima")/100000*100,"0.0")&"% meta")
Si ventas Lima ≥ 100K → "✓ Meta", si no → muestra el % alcanzado.
Patrón 3 — INDEX + MATCH + MATCH (búsqueda en 2D)
=INDEX(B2:E10, MATCH("Lima",A2:A10,0), MATCH("Enero",B1:E1,0))
Encuentra el valor en la intersección de fila "Lima" y columna "Enero" — como buscar en una tabla cruzada.
Cómo leer fórmulas anidadas sin perderte
Léelas de adentro hacia afuera. El resultado de la función más interna es el input de la siguiente. En el Patrón 3: primero los dos MATCH encuentran las posiciones, luego INDEX devuelve el valor en esa intersección.
