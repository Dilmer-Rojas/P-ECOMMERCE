Etapa 3 · Recolección de datos

El analista documenta de dónde viene el dato, cómo se obtiene y qué forma tiene antes de procesarlo. Un dato sin origen conocido no es confiable.

Fuente del dataset

📦

Análisis de Ventas para E-commerce (BI Consulting) — Kaggle

Este dataset fue diseñado por Regina N. Molares como parte de un trabajo práctico integrador para un curso de Business Intelligence con herramientas como Google Sheets y Looker Studio. Simula el funcionamiento de una empresa de e-commerce, permitiendo a los estudiantes aplicar conceptos de BI a través del análisis de transacciones, comportamiento de clientes y rendimiento de productos.

Carga del dataset en Python

# Paso 1 — importar librerías esenciales
import pandas as pd

# Paso 2 — cargar los archivos
rutas = {
    'clientes': '../data/raw/clientes.csv',
    'categorias': '../data/raw/categorias.csv',
    'productos': '../data/raw/productos.csv',
    'ventas': '../data/raw/ventas.csv',
    'metodos_pago': '../data/raw/metodos_pago.csv'
}

data = {nombre: pd.read_csv(ruta) for nombre, ruta in rutas.items()}

clientes, categorias, productos, ventas, metodos_pago = data.values()
